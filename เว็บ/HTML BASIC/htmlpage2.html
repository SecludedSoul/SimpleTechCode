1.PC or Notebook ได้เกือบทุกรูปแบบซึ่งหลายคนเข้าใจว่า “เห้ย! คอมพิวเตอร์สำหรับเขียนเว็บ หรือการCodingเนี่ยมันต้องแบบ เทพแน่เลย”

ผมขอบอกไว้ ณ ตรงนี้เลยน่ะครับ คือ คุณไม่จำเป็นที่จะต้องมีคอมแบบระดับ Server หรือคอมพิวเตอร์ระดับ HI-ENDที่เป็นเหมือนดังรูปคุณก็สามารถทำฺBasicเว็บและCodingได้เหมือนกัน

เอิ่ม...ซึ่งอันนี้มันก็เป็นคอมของผมเองแหละ แต่ทุกท่านไม่จำเป็นต้องลงทุนขนาดนี้น่ะครับ มีมันก็ดีเเหละเเต่อย่าเอาให้มันเกินความจําเป็นดีกว่า ในส่วนคอมของผมนี้มีงานrenderที่ค่อนข้างหนักเลยต้องลงทุนกันสักหน่อย ก็เพื่องานเน๊าะเราไม่เสียดายเงินหรอก T_T

หากคุณไม่รู้ว่าคอมตัวเองนั้นมันจะสามารถรองรับงานนี้ไหวหรือป่าวให้ลองทำความเข้าใจกับหัวข้อย่อยๆที่ผมจะอธิบายต่อไปนี้น่ะครับ

*หากท่านไหนไม่รู้ว่าคอมตัวเองนั้นมันจะสามารถรองรับงานนี้ไหวหรือป่าวและไม่รู้ในเรื่องสเปคคอม ให้ลองทำความเข้าใจกับหัวข้อย่อยๆที่ผมจะอธิบายต่อไปนี้น่ะครับ

-ขั้นต่ำอยากจะให้เป็นระบบปฏิบัติการที่เป็นWindows 64 Bitน่ะครับเพื่อความสเถียรของโปรแกรมโดยสามารถเชคได้ โดยการเปิด Setting ของ Windows และเข้าไปในเมนู About แล้วเชคได้ด้วยวิธีดังรูปได้เลย

-หากคอมคุณเขียน Word ทำ Powerpoint ได้ คุณก็สามารถCodding ทำWebPage เป็นของตัวเองได้แล้วครับ

-หากคอมคุณเป็นระบบปฏิบัติการ Mac OSหรือ Linuxผมแนะนำว่าให้คุณเข้าไปเช็คข้อมูลในลิงค์นี้ คลิกที่นี้!

2. โปรแกรมหรือSoftwareที่ไว้ใช้ในการCodingหรือทําWebSite ของเรา

หากท่านมีโปรแกรมหรือ Code Editor ในใจของท่านแล้วท่านก็สามารถใช้โปรแกรมของท่านกันได้เลยแต่หากอ้างอิงจากการสอนของผม ผมจะใช้ VsCode ซึ่งลิงค์การโหลดท่านสามารถ คลิกได้ที่นี้! และถ้าหากท่านอยากรู้วิธีการโหลดอย่างละเอียดและวิธีการใช้งานเบื้องต้น ได้ที่นี้! ขอบคุณคลิปสอนดีๆจาก AppzStory Studio อย่าลืมไปกดติดตามเเละสนับสนุนคุณ AppzStory Studio ได้ที่นี้!
