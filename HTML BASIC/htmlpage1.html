HOW to Use BASIC HTML & CSS

เรามาทำความรู้จักกับHTML และ CSS กันก่อนเลยล่ะกันน่ะครับเพื่อที่จะได้เข้าและรู้จักกับหน้าที่ของพวกมันกัน

Html เป็นภาษาที่ใช้ในการสร้างหรือใส่ contentต่างๆลงไปในweb page ไม่ว่าจะเป็นการ ใส่picture , Links หรือ text เป็นต้น เหมือนดังรูปเลย

แต่การที่เราใช้แต่ Htmlเนี่ยรูปร่างหรือความสวยงามของเว็บเนี่ย ผมบอกเลยไม่ผ่าน!

ดังนั้นเราจึงต้องใช้ cssเข้ามาช่วย ซึ่งพูดง่ายๆ CSS จะมีหน้าที่คอยช่วยตกแต่งเว็บเพจเราให้สวยขึ้นนั้นเอง เหมือนดังรูปเลย
